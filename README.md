# assemblyscript-demo

## Quick Start

### Install

```sh
git clone git@github.com:chenshenhai/assemblyscript-demo.git

cd assemblyscript-demo

npm i

npm run build
```

### Nodejs

```sh
npm run example:node
```

```sh
npm run example:browser
```
